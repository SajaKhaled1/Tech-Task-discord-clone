# Discord Clone - Real Time Chat App 💬

## 📖 Project Overview
This project is a simplified clone of Discord. The main goal was to build a real-time chat application that allows users to communicate inside different channels.

The app allows users to:
- Create an account and log in
- Join different chat channels
- Send and receive messages in real time
- Use a simple and user-friendly interface

## ⚙️ Tech Stack
- Node.js
- Express.js
- Socket.io
- MongoDB
- HTML, CSS, JavaScript
- bcryptjs (for password hashing)
- CORS

## ✨ Features
- User authentication (register / login)
- Real-time messaging using WebSockets
- Channel-based chat system
- Secure password encryption
- Clean and simple UI

## 🧠 List of Skills Used

- ⚙️ Building REST APIs using Express.js for handling backend logic and requests  
- 🍃 Working with MongoDB to store and manage users, messages, and chat data  
- 💬 Using Socket.io for real-time communication between users  
- 🔐 Implementing authentication and authorization for secure login and access control  
- 🔑 Password hashing using bcryptjs to securely store user passwords  
- 🎨 Frontend development using HTML, CSS, and JavaScript  
- 🌐 Understanding WebSockets for real-time data exchange between client and server  
- 🧩 Managing a full-stack project by connecting frontend with backend  
- 🚀 Deploying the project and uploading it to GitHub
